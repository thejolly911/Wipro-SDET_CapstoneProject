# Wipro SDET Capstone Project - Automation Framework

## Project Overview
This repository contains the automated test suite developed for the Capstone Project.

## Tech Stack
* **Language:** Java
* **Automation Tool:** Selenium WebDriver
* **Test Framework:** TestNG
* **Build Tool:** Maven

## How to Run the Tests
1. Clone the repository.
2. Open the project in Eclipse.
3. Right-click on `testng.xml` and select **Run As** -> **TestNG Suite**.
